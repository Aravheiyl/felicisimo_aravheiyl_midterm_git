Answer the following questions briefly:

1. What is Git used for?
Git is a version control system that is used to track changes in files over time, it also allow multiple people to collaborate, and manage different lines of development without overwriting each other's work.

2. What is the difference between Git and GitHub?
Git is the actual version control software that runs locally on the machine to track changes. GitHub is a website where all Git repositories are there, Github store our repo online and collaborate with others.

3. Why is branching useful?
Branching lets us work on new features and fix codes without affecting the main codebase. Developers can code and even make mistakes safely, then merge all the changes back into the main branch.

4. What command did you use to upload your local commits to GitHub?
